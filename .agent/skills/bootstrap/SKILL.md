# Bootstrap Skill

This skill helps you bootstrap a new Starlake project.

## Usage

```bash
starlake bootstrap [options]
```

## Options

- `--template <value>`: Template to use (default: quickstart)
- `--no-exit`: Should we exit after project creation?

## Examples

### Bootstrap Default

```bash
starlake bootstrap
```

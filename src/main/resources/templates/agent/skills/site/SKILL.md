# Site Skill

This skill generates a documentation website for your Starlake project.

## Usage

```bash
starlake site [options]
```

## Options

- `--outputDir <value>`: Output Directory
- `--template <value>`: Template name/path

## Examples

### Generate Documentation Site

```bash
starlake site --outputDir /path/to/site
```

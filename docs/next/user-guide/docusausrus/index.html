Documentation

To build your online documentation, you can use the starlake CLI to generate a static website from your markdown files.

starlake site --template docusaurus

Files will be generated in the metadata/site folder. You can set a different output folder with the --outputDir option.

You may use the default docusaurus template or create your own. See the starlake-site-docusaurus template for an example.
